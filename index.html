Grid System Documentation

The grid system provides a responsive, mobile-first layout system with customizable breakpoints, containers, rows, and columns.

Note: All grid classes are generated only if $enable-grid-classes is set to true (default).

Responsive Breakpoints

The grid system uses the following default breakpoints (can be customized via Sass variables):

Breakpoint Container Max Width CSS Variable
sm 540px --container-max-width-sm
md 720px --container-max-width-md
lg 960px --container-max-width-lg
xl 1140px --container-max-width-xl

The default gutter width is controlled by the CSS variable:

--grid-gutter-width: 30px;

Container Classes

Base Container Classes

Example:

<!-- Fluid container (100% width at all breakpoints) -->
<div class="container-fluid">
  <!-- Content -->
</div>

<!-- Responsive container (max-width at each breakpoint) -->
<div class="container">
  <!-- Content -->
</div>

Responsive Containers

These containers are 100% wide until the specified breakpoint:

Example:

<!-- 100% wide until the specified breakpoint, then max-width -->
<div class="container-sm">...</div>  <!-- max-width: 540px -->
<div class="container-md">...</div>  <!-- max-width: 720px -->
<div class="container-lg">...</div>  <!-- max-width: 960px -->
<div class="container-xl">...</div>  <!-- max-width: 1140px -->

Row Classes

Basic Row

Example:

<div class="row">
  <div class="col">Column 1</div>
  <div class="col">Column 2</div>
</div>

Row Variations

Examples:

<!-- Row with small gutters (16px) -->
<div class="row gutter-small">
  <div class="col">Column 1</div>
  <div class="col">Column 2</div>
</div>

<!-- Row without gutters -->
<div class="row no-gutters">
  <div class="col">Column 1</div>
  <div class="col">Column 2</div>
</div>

<!-- Row with bottom gutters (adds margin-bottom to columns) -->
<div class="row bottom-gutters">
  <div class="col">Column 1</div>
  <div class="col">Column 2</div>
</div>

Column System

The column system uses a 12-column layout with responsive breakpoints.

Basic Columns

Examples:

<div class="row">
  <div class="col">Auto-width column</div>
  <div class="col">Auto-width column</div>
</div>

<div class="row">
  <div class="col-6">Half-width column (6/12)</div>
  <div class="col-6">Half-width column (6/12)</div>
</div>

Responsive Columns

Example:

<!-- Stack on mobile, 50% on sm, 25% on md and up -->
<div class="row">
  <div class="col-12 col-sm-6 col-md-3">Column 1</div>
  <div class="col-12 col-sm-6 col-md-3">Column 2</div>
  <div class="col-12 col-sm-6 col-md-3">Column 3</div>
  <div class="col-12 col-sm-6 col-md-3">Column 4</div>
</div>

Available Column Classes

The system generates column classes for each breakpoint in the format:

  • col-{breakpoint}-{columns} (e.g., col-sm-6, col-md-4)
  • col-{columns} (extra small screens, no breakpoint)

Gutters

The default gutter width is 30px, controlled by the CSS variable:

--grid-gutter-width: 30px;

This can be modified:

/* Custom gutter width */
.row {
  --grid-gutter-width: 20px;
}

Customization

The grid system can be customized by modifying the Sass variables before importing the grid files:

// Example customization
$grid-breakpoints: (
  xs: 0,
  sm: 576px,
  md: 768px,
  lg: 992px,
  xl: 1200px
);

$container-max-widths: (
  sm: 540px,
  md: 720px,
  lg: 960px,
  xl: 1140px
);

$enable-grid-classes: true;

@import "grid";
Implementation Note: The grid system uses CSS variables for container max-widths (--container-max-width-{breakpoint}) and gutter width (--grid-gutter-width), making it easy to override these values dynamically.
