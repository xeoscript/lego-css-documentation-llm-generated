Margin & Padding Utility Classes

Spacing System

All spacing utilities are generated from a Sass map and use CSS custom properties (variables) for consistent spacing throughout your project.

Spacer Scale

The default spacer scale is based on a root value and uses a modular scale for consistent spacing:

0.25x (--spacer-1)
0.5x (--spacer-2)
1x (--spacer-3, base value)
1.5x (--spacer-4)
3x (--spacer-5)

Margin Utilities

Assign margin to all sides or specific sides of an element using responsive classes.

Basic Margin Classes

Format: .m{direction}-{size} where direction is optional and size corresponds to the spacer scale.

.m-{size}
margin: var(--spacer-{size}) !important;
Applies margin to all sides
.mt-{size}
margin-top: var(--spacer-{size}) !important;
Applies margin to top only
.mr-{size}
margin-right: var(--spacer-{size}) !important;
Applies margin to right only
.mb-{size}
margin-bottom: var(--spacer-{size}) !important;
Applies margin to bottom only
.ml-{size}
margin-left: var(--spacer-{size}) !important;
Applies margin to left only
.mx-{size}
margin-left: var(--spacer-{size}) !important; margin-right: var(--spacer-{size}) !important;
Applies margin to left and right
.my-{size}
margin-top: var(--spacer-{size}) !important; margin-bottom: var(--spacer-{size}) !important;
Applies margin to top and bottom

Auto Margin Classes

These classes set margin to auto, useful for centering or pushing elements to one side.

.m-auto
margin: auto !important;
Applies auto margin to all sides
.mt-auto
margin-top: auto !important;
Applies auto margin to top only
.mr-auto
margin-right: auto !important;
Applies auto margin to right only
.mb-auto
margin-bottom: auto !important;
Applies auto margin to bottom only
.ml-auto
margin-left: auto !important;
Applies auto margin to left only
.mx-auto
margin-left: auto !important; margin-right: auto !important;
Applies auto margin to left and right (horizontal centering)
.my-auto
margin-top: auto !important; margin-bottom: auto !important;
Applies auto margin to top and bottom

Responsive Spacing

All spacing utilities include responsive variants for each breakpoint in your grid system.

Breakpoint Suffixes

The responsive classes follow this pattern: .{property}{sides}-{breakpoint}-{size}

Breakpoint Infix Example Description
Extra small None .m-3 Applies to all screen sizes
Small -sm .m-sm-3 Applies to screens ≥576px
Medium -md .m-md-3 Applies to screens ≥768px
Large -lg .m-lg-3 Applies to screens ≥992px
Extra large -xl .m-xl-3 Applies to screens ≥1200px
Extra extra large -xxl .m-xxl-3 Applies to screens ≥1400px

Examples

Responsive Margin

Responsive Margin
<div class="m-2 m-md-4 m-lg-5"> Margin 2 on mobile, 4 on medium screens, 5 on large screens </div>

Responsive Padding

Responsive Padding
