Border Utility System

A comprehensive set of utility classes for borders, including width, color, and radius variations.

Note: All border utilities use CSS custom properties for easy theming and customization.

Border Width Classes

Control border width with these utility classes (values from 0.5px to 5rem):

All Borders

<!-- Apply to all borders -->
<div class="b-1">1px border</div>
<div class="b-2">2px border</div>
<div class="b-3">3px border</div>
<div class="b-4">4px border</div>
<div class="b-5">5px border</div>

Wrapper-Child Radius

Apply radius to first/last child elements within a container:

<div class="round-wc-3">
  <div class="b-2">First child gets top radius</div>
  <div class="b-2">Middle child</div>
  <div class="b-2">Last child gets bottom radius</div>
</div>
