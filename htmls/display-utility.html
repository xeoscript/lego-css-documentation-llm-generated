Responsive Display Classes

These classes allow you to change display properties at different breakpoints.

Breakpoint Suffixes

The responsive classes follow this pattern: .d-{breakpoint}-{property}

Breakpoint Infix Example Description
Extra small None .d-block Applies to all screen sizes
Small -sm .d-sm-block Applies to screens ≥576px
Medium -md .d-md-block Applies to screens ≥768px
Large -lg .d-lg-block Applies to screens ≥992px
Extra large -xl .d-xl-block Applies to screens ≥1200px
Extra extra large -xxl .d-xxl-block Applies to screens ≥1400px

Examples

.d-block .d-md-inline .d-lg-flex
display: block; @media (min-width: 768px) { display: inline; } @media (min-width: 992px) { display: flex; }

Implementation Code

These utilities are generated with SCSS:

// Basic display classes .d{ &-block{ display: block; } &-inline{ display: inline; } &-inline-block{ display: inline-block; } &-inline-flex{ display: inline-flex; } &-none{ display: none; } } // Responsive display utilities @each $breakpoint in map-keys($grid-breakpoints) { @include media-breakpoint-up($breakpoint) { $infix: breakpoint-infix-2($breakpoint, $grid-breakpoints); .d-#{$infix}none { display: none !important; } .d-#{$infix}block { display: block !important; } .d-#{$infix}inline { display: inline !important; } .d-#{$infix}inline-block { display: inline-block !important; } .d-#{$infix}flex { display: flex !important; } } }
