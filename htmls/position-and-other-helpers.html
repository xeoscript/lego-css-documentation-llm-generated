Position & Overflow Helper Classes

Absolute Positioning Classes

These classes position an element absolutely within its nearest positioned ancestor.

.abs-tl
position: absolute; top: 0; left: 0;
Positions element at top-left corner of its containing block
.abs-tr
position: absolute; top: 0; right: 0;
Positions element at top-right corner of its containing block
.abs-bl
position: absolute; bottom: 0; left: 0;
Positions element at bottom-left corner of its containing block
.abs-br
position: absolute; bottom: 0; right: 0;
Positions element at bottom-right corner of its containing block
.pos-a
position: absolute;
Sets position: absolute without any positioning (useful when you want to set custom values)

Examples

TR
BL

Relative Positioning Classes

.pos-r
position: relative;
Creates a new positioning context for absolutely positioned children

Example

This class is typically used as a parent for absolutely positioned elements.

<div class="pos-r"> <!-- Creates positioning context --> <div class="abs-tl">I'm positioned relative to my parent</div> </div>

Sticky Positioning Classes

These classes make elements stick to the viewport when scrolling.

.pos-st
position: sticky; top: 0;
Sticks to top of viewport when scrolling
.pos-sb
position: sticky; bottom: 0;
Sticks to bottom of viewport when scrolling

Example

Fixed Positioning Classes

These classes position elements relative to the viewport.

.pos-ft
position: fixed; top: 0; left: 0; right: 0;
Fixes element to top of viewport (full width)
.pos-fb
position: fixed; bottom: 0; left: 0; right: 0;
Fixes element to bottom of viewport (full width)

Example

Note: Fixed elements are removed from the normal document flow. These examples are simulated for demonstration.

Overflow Classes

These classes control how content behaves when it overflows its container.

.of-h
overflow: hidden;
Clips overflowing content (no scrollbars)
.of-v
overflow: visible;
Content is not clipped (may overflow container)
.of-s
overflow: scroll;
Adds scrollbars regardless of whether they're needed
.of-a
overflow: auto;
Adds scrollbars only when content overflows

Examples
