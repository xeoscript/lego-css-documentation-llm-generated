Flex Alignment System

A comprehensive flexbox utility system with responsive alignment classes for creating flexible layouts.

Note: All classes are prefixed with breakpoint variants (e.g., sm:fl-x, md:fl-a-c) for responsive control.

Core Flex Classes

Basic Flex Containers

Examples:

<!-- Horizontal flex container -->
<div class="fl-x">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

<!-- Vertical flex container -->
<div class="fl-y">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

Flex Directions

Class Property Description
fl-x flex-direction: row Horizontal layout (default)
fl-y flex-direction: column Vertical layout
fl-rr flex-direction: row-reverse Horizontal reversed
fl-cr flex-direction: column-reverse Vertical reversed

Predefined Combinations

Horizontal Alignment Combinations

Class Alignment Justification
fl-x-tl top (flex-start) left (flex-start)
fl-x-tc top (flex-start) center
fl-x-tr top (flex-start) right (flex-end)
fl-x-cl center left (flex-start)
fl-x-cc center center
fl-x-cr center right (flex-end)
fl-x-bl bottom (flex-end) left (flex-start)
fl-x-bc bottom (flex-end) center
fl-x-br bottom (flex-end) right (flex-end)

Example:

<!-- Centered both horizontally and vertically -->
<div class="fl-x-cc">
  <div>Centered Content</div>
</div>

<!-- Top-right aligned -->
<div class="fl-x-tr">
  <div>Top Right</div>
</div>

Responsive Breakpoints

All flex classes are available with responsive breakpoint prefixes:

Breakpoint Prefix Example
Extra small xs- xs-fl-x
Small sm- sm-fl-a-c
Medium md- md-fl-j-sb
Large lg- lg-fl-y-cc
Extra large xl- xl-fl-eq

Responsive Example:

<!-- Stack vertically on mobile, row on desktop -->
<div class="fl-y md-fl-x">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

<!-- Center items on large screens only -->
<div class="fl-x lg-fl-a-c">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

Additional Utilities

Class Description
fl-wrap Allow flex items to wrap
fl-no-wrap Prevent flex items from wrapping
fl-gr-1 to fl-gr-5 Flex grow values 1 through 5
fl-grow Flex item that grows to fill space
Tip: Combine these classes with your grid system for powerful responsive layouts.
