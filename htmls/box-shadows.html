Box Shadow Utilities

A comprehensive collection of box-shadow utilities with hover variants. All shadows include smooth transitions for hover states.

Basic Shadows

No shadow
.bs-none
box-shadow: none !important
Default shadow
.bs
box-shadow: 0 10px 40px 0 rgba(62, 57, 107, 0.07), 0 2px 9px 0 rgba(62, 57, 107, 0.06); transition: transform 0.2s ease-in-out, box-shadow 0.2s ease-in-out;
Light shadow
.bs-light
box-shadow: 0 2px 8px 0 rgba(50, 50, 50, .08)
Inset shadow
.bs-inset
box-shadow: inset 0 4px 7px 1px #fff, inset 0 -5px 20px rgba(173, 186, 204, .25), 0 2px 6px rgba(0, 21, 64, .14), 0 10px 20px rgba(0, 21, 64, .05)

All Shadow Classes

Each shadow class has a corresponding hover variant (e.g., .bs-1h for hover state of .bs-1).

Implementation Code

These utilities are generated with SCSS:

@mixin box-shadow($shadow) { box-shadow: $shadow; } .bs-none, .bs-none:hover { box-shadow: none !important } .bs { @include box-shadow((0 10px 40px 0 rgba(62, 57, 107, 0.07), 0 2px 9px 0 rgba(62, 57, 107, 0.06))); transition: transform 0.2s ease-in-out, box-shadow 0.2s ease-in-out; will-change: transform, box-shadow; &-light { @include box-shadow(0 2px 8px 0 rgba(50, 50, 50, .08)); } &-inset { @include box-shadow((inset 0 4px 7px 1px #fff, inset 0 -5px 20px rgba(173, 186, 204, .25), 0 2px 6px rgba(0, 21, 64, .14), 0 10px 20px rgba(0, 21, 64, .05))); } // ... and so on for all 68 shadow variations }
