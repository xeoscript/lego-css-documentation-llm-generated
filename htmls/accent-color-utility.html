Accent Color Utilities

Accent Color Classes

These classes set the accent-color property for form controls like checkboxes, radio buttons, and range inputs.

Each class has a hover/focus state that lightens the accent color for better visual feedback.

.accent-primary
accent-color: var(--color-primary) !important; &:hover, &:focus { accent-color: var(--color-primary-400) !important; }
Primary color
.accent-secondary
accent-color: var(--color-secondary) !important; &:hover, &:focus { accent-color: var(--color-secondary-400) !important; }
Secondary color
.accent-third
accent-color: var(--color-third) !important; &:hover, &:focus { accent-color: var(--color-third-400) !important; }
Third color
.accent-success
accent-color: var(--color-success) !important; &:hover, &:focus { accent-color: var(--color-success-400) !important; }
Success color
.accent-danger
accent-color: var(--color-danger) !important; &:hover, &:focus { accent-color: var(--color-danger-400) !important; }
Danger color
.accent-info
accent-color: var(--color-info) !important; &:hover, &:focus { accent-color: var(--color-info-400) !important; }
Info color
.accent-warning
accent-color: var(--color-warning) !important; &:hover, &:focus { accent-color: var(--color-warning-400) !important; }
Warning color

Examples

Checkboxes

.accent-primary
.accent-secondary
.accent-third
.accent-success
<input type="checkbox" class="accent-primary" checked> <input type="checkbox" class="accent-secondary" checked> <input type="checkbox" class="accent-third" checked> <input type="checkbox" class="accent-success" checked>

Radio Buttons

.accent-danger
.accent-info
.accent-warning
<input type="radio" name="radioGroup" class="accent-danger" checked> <input type="radio" name="radioGroup" class="accent-info"> <input type="radio" name="radioGroup" class="accent-warning">

Range Inputs

.accent-primary
.accent-success
.accent-danger
<input type="range" class="accent-primary" min="0" max="100" value="50"> <input type="range" class="accent-success" min="0" max="100" value="50"> <input type="range" class="accent-danger" min="0" max="100" value="50">

Hover/Focus States

Hover or focus on these elements to see the color change to the 400 variant:

.accent-info
.accent-warning
.accent-third
/* No need for extra code - hover/focus states are built into the classes */ <input type="checkbox" class="accent-info" checked> <input type="radio" class="accent-warning" checked> <input type="range" class="accent-third">

Implementation Code

// SCSS implementation @each $color, $value in $theme-colors { .accent-#{$color} { accent-color: var(--color-#{$color}) !important; &:hover, &:focus { accent-color: var(--color-#{$color}-400) !important; } } }
